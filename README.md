# 🧮 Calculadora Simples

Este é um projeto de **calculadora** desenvolvido com **Next.js**, **React** e estilizado com **Tailwind CSS**. A aplicação permite realizar operações matemáticas básicas de forma rápida e intuitiva.

## 🚀 Tecnologias utilizadas

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## 🛠 Funcionalidades

✔ Soma (+)  
✔ Subtração (-)  
✔ Multiplicação (×)  
✔ Divisão (÷)  

## 📦 Como rodar o projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/laurasrocha/calculadora.git

2. Acesse a pasta do projeto:
  cd calculadora

3. Instale as dependências:
  npm install

4. Inicie o servidor de desenvolvimento:
  npm run dev
